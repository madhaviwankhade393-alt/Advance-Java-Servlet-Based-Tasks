# PhoneBookApplication

A Phone Book Management System built using Java Servlets, JSP, JDBC, and MySQL following the MVC architecture.

## ✨ Features

* Add Contact
* View Contacts
* Search Contact
* Update Contact
* Delete Contact

## 🛠️ Technologies

* Java
* Servlets & JSP
* JDBC
* MySQL
* Apache Tomcat
* MVC Architecture

## 🎯 Purpose

Developed to gain hands-on experience in building dynamic web applications using Advanced Java technologies.
